3.9 查看系统信息

  显示操作系统的内核版本号。

uname -a


图3.9 1显示内核版本号

  查看系统主机名。

cat /etc/hostname


图3.9 2 查看系统主机名

  查看系统登录开机信息，（备注：非自动登录时会打印开机信息）。

cat /etc/issue


图3.9 3 查看系统登录欢迎信息

  查看CPU相关信息。

cat /proc/cpuinfo


图3.9 4 查看CPU相关信息

  查看内存相关信息。

cat /proc/meminfo


图3.9 5 查看内存相关信息
