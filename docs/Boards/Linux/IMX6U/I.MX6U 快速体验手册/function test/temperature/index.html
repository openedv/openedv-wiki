3.10 温度传感器

  I.MX6U芯片内部内置有温度传感器，可以实时反映I.MX6U CPU的内部温度

cat /sys/class/thermal/thermal_zone0/temp


图3.10 1 查看芯片内部温度

  温度值即为 51.440℃（51440/1000）。
